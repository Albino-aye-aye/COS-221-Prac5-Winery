The application uses my personal wheatley database. However, each group member made use of the same SQL queries to create tables on our own wheatley pages that are identical in structure.

This section deals with updating a user's credentials.
It makes use of cookies to keep track of which user is currently logged in and active, and based on that, the relevant user's credentials will be updated accordingly.
